üöÄ Quick Start

git clone https://github.com/alien-c0de/ip-intel-pro.git
cd ip-intel-pro
pip install -r requirements.txt
python main.py -s 8.8.8.8

Bulk Analysis: Create a text file with IP addresses (one per line) and run:

python main.py -i ip_list.txt

The analysis generates HTML, PDF, and CSV reports with comprehensive threat intelligence data.
